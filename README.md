# Torrent Name Parser
[![crates.io](https://img.shields.io/crates/v/torrent-name-parser.svg)](https://crates.io/crates/torrent-name-parser)

This is a small crate to parse metadata from a torrent's name.

Including but not limited to:

* Title
* Year
* Season
* Episode
* Codec
* Audio
* Resolution
* Quality
